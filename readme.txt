实验器材:
	探索者STM32F407开发板
	
实验目的:
	学习使用FreeRTOS互斥信号量的相关API函数。
	
硬件资源:
	1,DS0(连接在PB1)，DS1(连接在PB0上)
	2,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面) 
	3,ALIENTEK 2.8/3.5/4.3/7寸LCD模块(包括MCU屏和RGB屏,都支持) 

实验现象:
	在实验14-3中使用二值信号量的时候出现了优先级翻转的现象，本实验
	改用互斥信号量，缓解了优先级翻转的发生。
